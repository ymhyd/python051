### 五、项目介绍
项目编号：python051

宿舍故障报修系统是为高校等宿舍场景设计，支持学生线上提交故障报修、跟踪处理进度，管理员统筹派工、管理流程及反馈评价，实现故障高效处理与宿舍运维数字化的平台 。

该基于 Python 的宿舍故障报修系统，管理员可在 “评价反馈管理” 模块，对学生提交的报修评价反馈进行搜索、修改、删除，以及查看详情、回复审核 。
